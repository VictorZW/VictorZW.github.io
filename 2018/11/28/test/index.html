1.在GitHub上Create a new repository

2.Repository name 填写 yourname.github.io (yourname与你的注册用户名一致, 这个就是你博客的域名了)

二. 环境安装

1.安装node

node官网

2.安装git

3.安装Hexo

三. 博客搭建

1.进入到项目根目录

2.成功后执行

3.在浏览器输入 http://localhost:4000/就可以看见网页和模板了

4.接下来，把网页发布到GitHub上

5.配置博客
